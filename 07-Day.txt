break and continue:

continue: to skip the current iteration and cotinue from next 

list and its operation 
==============================
List	: is collection of data 
	: list is mutable(data can be changed)
	: list index start from 0

	: there are following methods
	 len()		: return count of element
	 max()		: return highest value
	 min()		: return lowest value
	 sum()		: return total
	 sort()		: arrange in asc order
	 reverse()	: print list from last to first
	 copy()		: make new copy of existing list
	 extend()	: merge one list into another list
	 
	 append()	: add new value at end
	 pop()		: remove last value
	 insert()	: add new value at given position 
	 remove()	: find the given and remove if exist
	 count()	: return count of given value

	 	 

	  slicer	: return data from given index
